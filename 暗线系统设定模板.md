# NPC暗线/后台世界推演系统提示词（System Prompt）

为了让插件能够完美提取并渲染好看的暗线UI，请将以下内容添加到你的 **系统提示词 (System Prompt)**、**作者说明 (Author's Note)** 或者 **世界书 (World Info)** 中，强制要求 AI 在正文后输出 Markdown 格式的表格。

---

## 推荐提示词

```markdown
### 【后台世界与NPC暗线推演法则】
在每次回复用户后，你必须在正文的最后，推演当前场景中未出场NPC的动态、或者敌对势力的阴谋、或者周围环境的暗中变化。
请务必使用 `<PHT>` 和 `</PHT>` 标签将推演内容包裹起来（标签名需与插件设置中保持一致）。
在标签内部，你**必须严格使用 Markdown 表格**的格式进行输出，表头必须包含：角色/势力、当前位置、隐藏动态、心理活动。表格简介不允许冗长。

**输出格式示例（请严格遵守此结构）：**
正文内容...对话和描写...

<PHT>
| 角色/势力 | 当前位置 | 隐藏动态 | 心理活动 |
| :--- | :--- | :--- | :--- |
| 李护卫 | 府邸后院密室 | 正在翻找那本丢失的账册，动作急躁 | “如果被老爷发现，我就死定了...” |
| 敌对帮派刺客 | 城外破庙 | 正在擦拭淬毒的暗器，等待夜幕降临 | 迫不及待想要看到目标倒在血泊中 |
</PHT>
```

---

## 使用说明：
1. **统一标签名**：如果你在上方设定中修改了 `<PHT>` 标签名，请确保在插件设置界面的【目标标签】处填写同样的名称。
2. **列的动态扩展**：你可以随意更改表格的列名（比如改成 `|姓名|好感度|下一步计划|`），插件的 UI 渲染系统会自动适配并生成精美的卡片式编辑界面！
3. **编辑与修改**：当 AI 输出带有表格的标签后，只需点击顶部栏的钢笔图标（ST-Text-Replacer 插件），在【暗线面板】中即可看到排版极其精美的卡片。你可以直接在卡片上修改这些设定，点击保存后将直接同步到聊天记录中！
